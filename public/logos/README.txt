Положите сюда единый логотип IBG. Поддерживаемые имена (проверяются в этом порядке):

  ibg.svg   — предпочтительно (векторное, идеально масштабируется)
  ibg.png
  ibg.jpg / ibg.jpeg
  ibg.webp

Файл встраивается в PDF как base64 data-URI (см. src/lib/pdf/template.ts
→ loadUniversalLogoDataUri). Кэш модульного уровня — сервер читает диск
один раз на холодный старт.

Рекомендуемые размеры:
  - Растровые: высота 200–400px, ширина до 1200px (в PDF ужмётся до 46pt)
  - Прозрачный фон для растра — обязателен

Если файла нет, в шапке PDF остаётся текстовый placeholder с названием компании.
